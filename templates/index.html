📄 DocuMint – Create Word Docs from Excel, in Bulk

Generate individual Word documents in bulk from an Excel spreadsheet. Ideal for contracts, letters, tenancy agreements, and more.

DocuMint is a web-based tool that creates one Word document per row in your Excel file, using a single Word template. Unlike Microsoft Word’s Mail Merge, which creates one long file with multiple pages, DocuMint generates and names each file separately — then gives you a single ZIP file to download.

Placeholders like «First_Name» or «Start_Date» inside your Word template will be replaced with actual data from your spreadsheet. You can type these manually or use Word’s Mail Merge fields. DocuMint supports both — including inside tables and paragraphs.

📊 Step-by-Step Instructions

1. Create Your Excel File

Use column headers that match your placeholder names. Each row will generate one document.

2. Create or Open Your Word Template

Use «Placeholders» in your document where you want data to appear. Use underscores instead of spaces (e.g. «First_Name»).

You can also use Microsoft Word's Mail Merge feature and insert fields using the “Insert Merge Field” tool.

3. Upload and Process

DocuMint will generate one Word document for each row of your Excel file and zip them into a single download.

✅ Tip:

  • Make sure your Excel column names match your placeholder names (replace spaces with underscores).
  • You can include placeholders inside tables and paragraphs.
  • Files are not stored — they’re deleted immediately after download.
